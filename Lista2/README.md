**Exercício 5**:

Crie uma função chamada `calcularMedia` que recebe um array de números como parâmetro e retorna a média dos números.

```
console.log(calcularMedia([1,2,3,4,5]));// 3
```

**Exercício 7**:

Crie uma função que aceite dois parâmetros: um array de números e um número. A função deve retornar um novo array com todos os números maiores que o número fornecido.

Exemplo de uso:

```
letarray= [1,2,3,4,5,6,7,8,9];
letnum=5;
console.log(filtrarNumeros(array,num));// [6, 7, 8, 9]
```

**Exercício 8**:

Crie uma função que aceite um array e um valor. A função deve retornar a quantidade de vezes que o valor aparece no array.

Exemplo de uso:

```
letarray= [1,2,2,3,3,3,4,4,4,4];
letvalor=4;
console.log(contarOcorrencias(array,valor));// 4
```

**Exercício 10**:

Crie uma função que aceite um número e retorne um array com todos os números de 1 até o número fornecido, incluindo o número fornecido.

letnumero=5;
console.log(criarArray(numero));// [1, 2, 3, 4, 5]

**Exercício 11**:

Crie um objeto chamado `carro` com propriedades: `marca`, `modelo` e `ano`.

1. Acesse a propriedade `marca` do objeto `carro`.
2. Altere a propriedade `ano` do objeto `carro` para 2025.
3. Adicione um método ao objeto `carro` chamado `getIdade` que retorne quantos anos o carro tem (ano atual - ano do carro).
4. Adicione um método ao objeto `carro` chamado `getDescricao` que retorne uma string contendo todas as informações do carro.

**Exercício 15**:

Crie um array chamado `frutas` que contém "maçã", "banana" e "laranja".

1. Imprima o segundo elemento do array `frutas`.
2. Adicione "manga" ao final do array `frutas`.
3. Remova o primeiro elemento do array `frutas`.
4. Verifique o tamanho do array `frutas`.
5. Crie um loop `for` que percorre o array `frutas` e imprime cada fruta.
6. Use o método `forEach` para imprimir cada elemento do array `frutas`.
7. Use o método `map` para criar um novo array que contém o tamanho de cada fruta do array `frutas`.
8. Use o método `filter` para criar um novo array que contém apenas as frutas do array `frutas` que têm mais de 5 caracteres.
9. Use o método `reduce` para calcular a soma dos números em um array.
